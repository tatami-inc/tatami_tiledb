Parameters
uriFile path (or some other appropriate location) of the TileDB array.
attributeName of the attribute containing the data of interest.
optionsFurther options.

Create a tatami::Matrix representation from a 2-dimensional dense or sparse TileDB array. This function will automatically choose the most appropriate representation based on the storage format. For dense on-disk arrays, a TileDbDenseMatrix is constructed; otherwise a TileDbSparseMatrix is returned.
