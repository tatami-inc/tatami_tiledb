tatami for TileDB matrices

This repository implements tatami bindings for TileDB-backed matrices, allowing random access without loading the entire dataset into memory. Matrices can be stored as 2-dimensional TileDB datasets:

#include "tatami_tiledb/tatami_tiledb.hpp"
tatami_hdf5::TileDbDenseMatrix<double, int> dense_mat("some_file.h5", "dataset_name");

Check out the reference documentation for more details.
