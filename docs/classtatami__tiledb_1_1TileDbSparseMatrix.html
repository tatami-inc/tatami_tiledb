Detailed Description

template<typename Value_, typename Index_, bool transpose_ = false>
class tatami_tiledb::TileDbSparseMatrix< Value_, Index_, transpose_ >

TileDB-backed sparse matrix.

Template Parameters
Value_Numeric type of the matrix value.
Index_Integer type for the row/column indices.
transpose_Whether to transpose the on-disk data upon loading. By default, this is false, so the first dimension corresponds to rows and the second dimension corresponds to columns. If true, the first dimension corresponds to columns and the second dimension corresponds to rows.

Numeric sparse matrix stored in a 2-dimensional TileDB array. Chunks of data are loaded from file as needed by tatami::Extractor objects, with additional caching to avoid repeated reads from disk. The size of cached chunks is determined by the extent of the tiles in the TileDB array.
